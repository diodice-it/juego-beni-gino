# Carrera Beni vs Gino

Juego HTML de carrera infinita para 2 jugadores (por turnos). Esquivá obstáculos saltando o agachándote. Gana el que recorre más metros.

## Cómo jugar

1. Cada jugador elige su skin
2. Primero corre Gino, después Beni
3. Controles:
   - **Saltar**: Espacio / Flecha arriba / W
   - **Agacharse**: Flecha abajo / S
   - **Celular**: Tocar arriba = saltar, tocar abajo = agacharse
4. Un golpe y perdés
5. Al final se comparan las distancias

## Skins disponibles

Boca, Huracán, Superhéroe, Ninja, Astronauta, Zombie, Rey, Rockero

## Escenarios

Los escenarios cambian automáticamente cada 1000m:

- **Bosque** - troncos, hongos, pájaros
- **Desierto** - cactus, serpientes, buitres
- **Nieve** - bloques de hielo, muñecos de nieve
- **Ciudad** - autos, conos, carteles
- **Espacio** - meteoritos, aliens, satélites

La velocidad aumenta progresivamente.

## Tech

HTML + CSS + JavaScript puro (Canvas). Un solo archivo `index.html`, sin dependencias.
